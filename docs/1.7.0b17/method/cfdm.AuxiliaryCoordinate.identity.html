cfdm.AuxiliaryCoordinate.identity¶

AuxiliaryCoordinate.identity(default='')[source]¶

Return the canonical identity.

By default the identity is the first found of the following:

  1. The “standard_name” property.
  2. The “cf_role” property, preceeded by 'cf_role='.
  3. The “axis” property, preceeded by 'axis='.
  4. The “long_name” property, preceeded by 'long_name='.
  5. The netCDF variable name, preceeded by 'ncvar%'.
  6. The identity of the bounds, if any.
  7. The value of the default parameter.

New in version 1.7.0.

See also

identities

Parameters:
default: optional

If no identity can be found then return the value of the default parameter.
Returns:

The identity.

Examples:

>>> f.properties()
{'foo': 'bar',
 'long_name': 'Air Temperature',
 'standard_name': 'air_temperature'}
>>> f.nc_get_variable()
'tas'
>>> f.identity()
'air_temperature'
>>> f.del_property('standard_name')
'air_temperature'
>>> f.identity(default='no identity')
'air_temperature'
>>> f.identity()
'long_name=Air Temperature'
>>> f.del_property('long_name')
>>> f.identity()
'ncvar%tas'
>>> f.nc_del_variable()
'tas'
>>> f.identity()
'ncvar%tas'
>>> f.identity()
''
>>> f.identity(default='no identity')
'no identity'

>>> f.properties()
{}
>>> f.bounds.properties()
{'axis': 'Z',
 'units': 'm'}
>>> f.identity()
'axis=Z'
