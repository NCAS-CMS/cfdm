cfdm.RaggedIndexedArray.ndim¶

RaggedIndexedArray.ndim¶

The number of dimensions of the uncompressed data.

Examples:

>>> d.shape
(73, 96)
>>> d.ndim
2
>>> d.size
7008

>>> d.shape
(1, 1, 1)
>>> d.ndim
3
>>> d.size
1

>>> d.shape
()
>>> d.ndim
0
>>> d.size
1
