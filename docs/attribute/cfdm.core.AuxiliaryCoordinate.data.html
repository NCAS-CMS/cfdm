cfdm.core.AuxiliaryCoordinate.data¶

AuxiliaryCoordinate.data¶

Return the data.

f.data is equivalent to f.get_data()

Note that a Data object is returned. Use the get_array method to return the data as a numpy array.

New in version 1.7.

See also

del_data, get_array, get_data, has_data, set_data

Returns:
out: Data

The data.

Examples:

>>> d = cfdm.Data(range(10))
>>> f.set_data(d)
>>> f.has_data()
True
>>> d = f.data
>>> d
<Data: [0, ..., 9]>
>>> f.data.shape
(10,)
