cfdm.core.DimensionCoordinate.bounds¶

DimensionCoordinate.bounds¶

Return the bounds.

f.bounds is equivalent to f.get_bounds()

Note that a Bounds instance is returned. Use the get_array method of the Bounds instance to return the bounds as a numpy array.

New in version 1.7.

See also

data, del_bounds, get_bounds, has_bounds, set_bounds

Returns:
out: Bounds

The bounds.
