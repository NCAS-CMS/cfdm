cfdm.core.Array.shape¶

Array.shape¶

Tuple of array dimension sizes.

New in version 1.7.0.

Examples:

>>> a.shape
(73, 96)
>>> a.ndim
2
>>> a.size
7008

>>> a.shape
(1, 1, 1)
>>> a.ndim
3
>>> a.size
1

>>> a.shape
()
>>> a.ndim
0
>>> a.size
1
