cfdm.CellConnectivity.data¶

CellConnectivity.data¶

The data.

f.data is equivalent to f.get_data().

Note that a Data instance is returned. Use the array attribute to get the data as a numpy array.

The units, calendar and fill value properties are, if set, inserted into the data.

Added in version (cfdm): 1.7.0

See also

cfdm.Data.array, del_data, get_data, has_data, set_data

Returns:
Data

The data.

Examples

>>> f = cfdm.CellConnectivity()
>>> f.set_data(cfdm.Data(numpy.arange(10.)))
>>> f.has_data()
True
>>> d = f.data
>>> d
<Data(10): [0.0, ..., 9.0]>
>>> f.data.shape
(10,)
