cfdm.core.Data.set_calendar¶

Data.set_calendar(value)[source]¶

Set the calendar.

See also

del_calendar, get_calendar, has_calendar, set_units

Parameters
value: str

The new calendar.

Returns

None

Examples:

>>> d.set_calendar('360_day')
>>> d.has_calendar()
True
>>> d.get_calendar()
'360_day'
>>> d.del_calendar()
>>> d.has_calendar()
False
>>> d.get_calendar()
ValueError: Can't get non-existent calendar
>>> print(d.get_calendar(None))
None
>>> print(d.del_calendar(None))
None
