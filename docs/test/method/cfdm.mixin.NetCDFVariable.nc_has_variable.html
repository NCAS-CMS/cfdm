cfdm.mixin.NetCDFVariable.nc_has_variable¶

NetCDFVariable.nc_has_variable()[source]¶

Whether the netCDF variable name has been set.

New in version 1.7.

See also

nc_del_variable, nc_get_variable, nc_set_variable

Returns:
out: bool

True if the netCDF variable name has been set, otherwise False.

Examples:

>>> f.nc_set_variable('tas')
>>> f.nc_has_variable()
True
>>> f.nc_get_variable()
'tas'
>>> f.nc_del_variable()
'tas'
>>> f.nc_has_variable()
False
>>> print(f.nc_get_variable(None))
None
>>> print(f.nc_del_variable(None))
None
