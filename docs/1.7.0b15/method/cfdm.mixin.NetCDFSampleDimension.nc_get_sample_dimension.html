cfdm.mixin.NetCDFSampleDimension.nc_get_sample_dimension¶

NetCDFSampleDimension.nc_get_sample_dimension(*default)[source]¶

Return the netCDF sample dimension name.

New in version 1.7.0.

See also

nc_del_sample_dimension, nc_has_sample_dimension, nc_set_sample_dimension

Parameters:
default: optional

Return default if the netCDF sample dimension name has not been set.
Returns:
str

The netCDF sample dimension name. If unset then default is returned, if provided.

Examples:

>>> f.nc_set_sample_dimension('time')
>>> f.nc_has_sample_dimension()
True
>>> f.nc_get_sample_dimension()
'time'
>>> f.nc_del_sample_dimension()
'time'
>>> f.nc_has_sample_dimension()
False
>>> print(f.nc_get_sample_dimension(None))
None
>>> print(f.nc_del_sample_dimension(None))
None
