cfdm.Constructs.axis¶

Constructs.axis(*axes)[source]¶

Select metadata constructs

New in version 1.7.0.

See also

key, measure, method, name, ncdim, ncvar, property, type

Parameters:
axes:

Select constructs that have data which spans at least one of the given domain axes constructs. Domain axes constructs are specified with their construct keys.

If an axis of None is provided then all constructs that have data are selected.
Returns:
Constructs

The selected constructs and their construct keys.

Examples:

>>> d = c.axis('domainaxis1')

>>> d = c.axis('domainaxis0', 'domainaxis1')

Setting no keyword arguments selects no constructs:

>>> c.key()
