cfdm.Field.set_data_axes¶

Field.set_data_axes(axes)[source]¶

Set the domain axes spanned by the data array.

New in version 1.7.

See also

data, del_data_axes, get_data, get_data_axes

Parameters:
axes: sequence of str

The identifiers of the domain axes spanned by the data array.

Example:

axes=['domainaxis0', 'domainaxis1']
Returns:

None
