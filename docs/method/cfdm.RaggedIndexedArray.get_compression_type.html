cfdm.RaggedIndexedArray.get_compression_type¶

RaggedIndexedArray.get_compression_type()[source]¶

The type of compression that has been applied to the underlying array.

New in version 1.7.

Returns:
out: str

The compression type. An empty string means that no compression has been applied.
Examples: 
>>> a.compression_type
''

>>> a.compression_type
'gathered'

>>> a.compression_type
'ragged contiguous'
