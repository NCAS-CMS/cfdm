cfdm.CompressedArray.get_filenames¶

CompressedArray.get_filenames()[source]¶

Return the names of any files containing the compressed data.

New in version (cfdm): 1.10.0.2

Returns
set

The file names in normalised, absolute form. If the data are all in memory then an empty set is returned.
