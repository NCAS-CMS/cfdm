cfdm.core.Data.get_calendar¶

Data.get_calendar(*default)[source]¶

Return the calendar.

See also

del_calendar, set_calendar

Parameters:
default: optional

Return default if calendar has not been set.
Returns:

The calendar. If calendar has not been set then return the value of default parameter, if provided.

Examples:

>>> d.set_calendar('julian')
>>> d.get_calendar
'metres'
>>> d.del_calendar()
>>> d.get_calendar()
AttributeError: Can't get non-existent calendar
>>> print(d.get_calendar(None))
None
