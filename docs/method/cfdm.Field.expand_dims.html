cfdm.Field.expand_dims¶

Field.expand_dims(axis, position=0)[source]¶

Expand the shape of the data array.

Insert a new size 1 axis, corresponding to a domain axis construct, into the data array.

New in version 1.7.

See also

squeeze, transpose

Parameters:
axis: str

The construct identifier of the domain axis construct corresponding to the inserted axis.

Example:

axis='domainaxis2'

position: int, optional

Specify the position that the new axis will have in the data array. By default the new axis has position 0, the slowest varying position. Negative integers counting from the last position are allowed.

Example:

position=2

Example:

position=-1
Returns:
out: Field

The new field construct with expanded data axes.

Examples:

>>> f.data.shape
(19, 73, 96)
>>> f.expand_dims('domainaxis3').data.shape
(1, 96, 73, 19)
>>> f.expand_dims('domainaxis3', position=3).data.shape
(19, 73, 96, 1)
>>> f.expand_dims('domainaxis3', position=-1).data.shape
(19, 73, 1, 96)
