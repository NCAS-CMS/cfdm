cfdm.Field.construct_axes¶

Field.construct_axes(cid=None)[source]¶

Return the identifiers of the domain axes spanned by the construct data arrays.

New in version 1.7.

See also

constructs, get_construct

Parameters:
cid: str

The identifier of the construct.
Returns:
out: tuple or None

The identifiers of the domain axes spanned by the construct’s data array. If the construct does not have a data array then None is returned.

Examples

>>> f.construct_axes('auxiliarycoordinate0')
('domainaxis1', 'domainaxis0')
>>> print(f.construct_axes('auxiliarycoordinate99'))
None
