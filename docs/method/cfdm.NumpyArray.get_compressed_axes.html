cfdm.NumpyArray.get_compressed_axes¶

NumpyArray.get_compressed_axes()[source]¶

Return axes that are compressed in the underlying array.

New in version 1.7.0.

Returns:
list

The compressed axes described by their integer positions in the list of uncompressed axes. If no axes have been compressed then the list is empty.
