cfdm.AuxiliaryCoordinate.identities¶

AuxiliaryCoordinate.identities()[source]¶

Return all possible identities.

The identities comprise:

  • The “standard_name” property.
  • All properties, preceeded by the property name and a colon, e.g. 'long_name:Air temperature'.
  • The netCDF variable name, preceeded by 'ncvar%'.
  • The identities of the bounds, if any.

New in version 1.7.0.

See also

identity

Returns:
list

The identities.

Examples:

>>> f.properties()
{'foo': 'bar',
 'long_name': 'Air Temperature',
 'standard_name': 'air_temperature'}
>>> f.nc_get_variable()
'tas'
>>> f.identities()
['air_temperature',
 'long_name=Air Temperature',
 'foo=bar',
 'standard_name=air_temperature',
 'ncvar%tas']

>>> f.properties()
{}
>>> f.bounds.properties()
{'axis': 'Z',
 'units': 'm'}
>>> f.identities()
['axis=Z', 'units=m', 'ncvar%z']
