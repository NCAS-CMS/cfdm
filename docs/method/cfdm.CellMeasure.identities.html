cfdm.CellMeasure.identities¶

CellMeasure.identities()[source]¶

Return all possible identities.

The identities comprise:

  • The measure property, preceeded by 'measure:'.
  • The “standard_name” property.
  • All properties, preceeded by the property name and a colon, e.g. 'long_name:Air temperature'.
  • The netCDF variable name, preceeded by 'ncvar%'.

New in version 1.7.0.

See also

identity

Returns:
list

The identities.

Examples:

>>> f.properties()
{'foo': 'bar',
 'long_name': 'Area of cells',
 'standard_name': 'cell_area'}
>>> f.nc_get_variable()
'areacello'
>>> f.identities()
['measure:area',
 'cell_area',
 'long_name=Area of cells',
 'foo=bar',
 'standard_name=cell_area',
 'ncvar%areacello']
