cfdm.mixin.PropertiesData.expand_dims¶

PropertiesData.expand_dims(position=0)[source]¶

Expand the shape of the data array.

Insert a new size 1 axis into the data array.

New in version 1.7.

See also

squeeze, transpose

Parameters:
position: int, optional

Specify the position that the new axis will have in the data array. By default the new axis has position 0, the slowest varying position. Negative integers counting from the last position are allowed.

Example:

position=2

Example:

position=-1
Returns:
out:

The new construct with expanded data axes.

Examples:

>>> f.data.shape
(19, 73, 96)
>>> f.expand_dims(position=3).data.shape
(19, 73, 96, 1)
>>> f.expand_dims(position=-1).data.shape
(19, 73, 1, 96)
