cfdm.CoordinateConversion.parameters¶

CoordinateConversion.parameters(parameters=None, copy=True)[source]¶

Return or replace all parameter-valued terms.

New in version 1.7.0.

See also

ancillaries

Parameters:
parameters: dict, optional

Delete all existing named parameters, and instead store those from the dictionary supplied.

Parameter example:

parameters={'earth_radius': 6371007}

Parameter example:

parameters={}

copy: bool, optional

If False then any parameters provided by the parameters parameter are not copied before insertion. By default they are deep copied.
Returns:
dict

The parameters or, if the parameters parameter was set, the original parmaeters.

Examples:

>>> c.parameters()
{'standard_parallel': 25.0;
 'longitude_of_central_meridian': 265.0,
 'latitude_of_projection_origin': 25.0}

>>> c.parameters()
{}
