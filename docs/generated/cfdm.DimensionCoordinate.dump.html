cfdm.DimensionCoordinate.dump¶

DimensionCoordinate.dump(display=True, _omit_properties=None, field=None, key=None, _level=0, _title=None, _axes=None, _axis_names=None)[source]¶

Return a string containing a full description of the auxiliary coordinate object.

Parameters:
display: bool, optional

If False then return the description as a string. By default the description is printed, i.e. f.dump() is equivalent to print f.dump(display=False).
Returns:
out: None or str

A string containing the description.
